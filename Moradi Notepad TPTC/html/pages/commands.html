  • NickServ
    • Register
      • /ns REGISTER Password EMAIL
      • Example: /ns REGISTER CoolDude Pa$$w0rd cooldude@ThePlaceToChat.net
    • Group
      • /ns GROUP NICK Password
      • Example: /ns GROUP CoolDude Pa$$w0rd
    • Identify
      • /ns ID Password
      • Example: /ns ID Pa$$w0rd
    • Drop
      • /ns DROP NICK
      • Example: /ns DROP CoolDude
    • Change Password
      • /ns SET PASSWORD NewPassword
      • Example: /ns SET PASSWORD Pa$$w0rd123
    • Recover Nick
      • /ns RECOVER NICK Password
      • Example: /ns RECOVER CoolDude Pa$$w0rd
    • Release Nick
      • /ns RELEASE NICK Password
      • Example: /ns RELEASE CoolDude Pa$$w0rd
      • (This will allow you to remove any holds on your nick not allowing you to use it)
    • Release Nick
      • /ns RELEASE NICK Password
      • Example: /ns RELEASE CoolDude Pa$$w0rd
      • (This will allow you to remove any holds on your nick not allowing you to use it)
    • Ghost Nick
      • /ns GHOST NICK Password
      • Example: /ns GHOST CoolDude Pa$$w0rd
      • (This will allow you to force your nick to disconnect from the server so you can use it)
    • Nick Info
      • /ns INFO NICK ALL
      • Example: /ns INFO CoolDude ALL
      • (This will display information about the nick such as: creation date, Host, Connected Channels, Servers etc..)
    • Glist
      • /ns GLIST
      • Example: /ns GLIST
      • (This will allow you to review all nicks in your nick group)
  • Chanserv
    • Register Channel
      • /cs REGISTER #Channel Password Description
      • Example: /cs REGISTER #lobby Pa$$word Welcome To Lobby
      • (This will allow you to register a chennel of your own on our network)
    • Identify Channel
      • /cs IDENTIFY Channel Password
      • Example: /cs ID #lobby Pa$$word
      • (This will allow you to Identify yourself as channel owner from a diff. nick)
    • Drop Channel
      • /cs DROP Channel
      • Example: /cs DROP #lobby
      • (This will allow you to unregister your channel on our server)
    • Set Founder
      • /cs SET #channel FOUNDER nick
      • Example: /cs SET #lobby FOUNDER DJREENO
      • (This will allow you to change the current founder of the channel)
    • Set Ban Type
      • /cs SET #Channel BANTYPE bantype
      • Example: /cs SET #lobby BANTYPE 0-3
      • 0: ban in the form *!user@host
      • 1: ban in the form *!*user@host
      • 2: ban in the form *!*@host
      • 3: ban in the form *!*user@*.domain
      • (This will allow you to set the default bantype for you channel)
    • Set Keep Topic
      • /cs SET #channel KEEPTOPIC {ON | OFF}
      • Example: /cs SET #lobby KEEPTOPIC ON
      • (This will allow you to Force your channel to keep the topic even if all chatters leave the channel)
    • Set Mlock
      • /cs SET #channel MLOCK +modes
      • Example: /cs SET #lobby MLOCK +nt
      • (This will allow you to force the channel to lock/keep the channel modes you set)
    • Set Peace
      • /cs SET #channel PEACE {ON | OFF}
      • Example: /cs SET #lobby PEACE ON
      • (This will allow you to disable channel OPS from kicking, banning other OPS with higher or equal ranking)
    • Set PRIVATE
      • /cs SET #channel PRIVATE {ON | OFF}
      • Example: /cs SET #lobby PRIVATE ON
      • (This will allow you to hide your channel from appearing in any room searches such as /list)
    • Set SIGNKICK
      • /cs SET #channel SIGNKICK {ON | LEVEL | OFF}
      • Example: /cs SET #lobby SIGNKICK OFF
      • (This will allow you to hide or show your nick when kicking users, LEVEL hides your nick only if you have a higher ranking)
    • AOP (AutoOP)
      • /cs AOP #channel {ADD | DEL | LIST | CLEAR} Nick
      • Example: /cs AOP #lobby ADD DJREENO
      • (This will allow you to add chatters to the auto OP list, they will be given OP automaticly when entering)
      • AOP is the highest channel admin under owner(sop), they can Voice/Devoice, Halfop/DeHalfOP, kick, ban, change channel modes
    • HOP (HalfOP)
      • /cs HOP #channel {ADD | DEL | LIST | CLEAR} NICK
      • Example: /cs HOP #lobby ADD DJREENO
      • (This will allow you to add chatters to the auto HALFOP list, they will be given HALFOP automaticly when entering)
      • HOP can kick,ban,voice,devoice users on a channel
    • SOP (SuperOP)
      • /cs SOP #channel {ADD | DEL | LIST | CLEAR} NICK
      • Example: /cs SOP #lobby ADD DJREENO
      • (This will allow you to add chatters to the auto SOP list, they will be given SOP automaticly when entering)
      • SOP is the highest channel admin, SOP can do everything! except drop the channel. Kick,Ban,Voice/Devoice,OP/Deop,Halfop/DeHalfop, Channel Modes, Channel Topic etc..
    • VOP (VoicePeople)
      • /cs VOP #channel {ADD | DEL | LIST | CLEAR} NICK
      • Example: /cs VOP #lobby ADD DJREENO
      • (This will allow you to add chatters to the auto VOP list, they will be given VOP automaticly when entering)
      • VOP allows users to chat in a channel that has MODE +m (Moderated Channel)
    • AKICK (AutoKick)
      • /cs AKICK #channel ADD {Nick | Mask} Reason
      • Example: /cs AKICK #lobby ADD DAZ Not Permited On This Channel, Goodbye
      • (This will allow you to add chatters to the auto KICK list, they will be KICKED & BANNED automaticly when entering)
      • AKICK Auto Kicks/Bans Users On Channel Entrance
    • Channel INFO
      • /cs INFO #channel ALL
      • Example: /cs INFO #lobby ALL
      • (This will allow you to view all info about a channel)
      • View Creation Date, owner, decription, bantype, etc..
    • OP, DEOP, VOICE, DEVOICE, HALFOP, DEHALFOP
      • /cs {OP | DEOP | VOICE | DEVOICE | HALFOP | DEHALFOP} #channel NICK
      • Example: /cs OP #lobby DJREENO
      • (This will allow you to Temp. give users channel permissions)
      • OP: Chanel Admin (kick, ban, voice, devoice, halfop, dehalfop)
      • HALFOP: Half Channel Admin (kick,ban, voice, devoice)
      • VOICE: (Allows chatter to speak on a channel that has mode +m set to pervent unauthorized chatters from chatting)
